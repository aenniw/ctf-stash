# CTF-Stash

## Building locally

To work locally with this project, you'll have to follow the steps below:

1. Fork, clone or download this project
1. Install Docsify `npm install docsify-cli -g`
1. Preview your project: `docsify serve`

## CTF dumps

```bash
python3 ./ctf-dump.py
```
